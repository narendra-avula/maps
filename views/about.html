About

It is a simple weekend project that I developed in order to quickly jolt my Angular JS knowlege of a few things.

What is it?

You can search a country and get several details regarding the country and check the map.

What is it made of?

The app uses the following:

  • v1.4.7 of Angular JS
  • ui-router is being used for routing.
  • Twitter Bootstrap for the interface.
  • Angular Google Maps for the map

What does it contain?

There are a few controllers, some nested views, a service, an HTTP Interceptor and a directive for the loader.

What is missing?

The dependencies should have been installed through bower, I should have wrote some gulp tasks for minifying and combining those numerous Javascript files. But I was Okay with it as long as it served the purpose and it did.
